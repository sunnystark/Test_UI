# React Login

> Implementation of Otka authorization/signin in React

Uses react-router-dom, @okta/okta-react and @okta/okta-signin-widget

## For Quick Start

``` bash
# Install dependencies
npm install

# Serve on localhost:3000
npm start

# Build for production
npm run build
```
